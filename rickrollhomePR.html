Rickroll Website

This is one of my fun, experimental projects. I built a simple website that rickrolls visitors. The project was a great way to practice HTML, CSS, and some JavaScript tricks.

Why I Made It

To practice front-end web development in a fun way.

To experiment with interactive buttons and redirects.

How It Works

The page loads a simple interface with a button.

Clicking the button redirects users to a Rickroll video.

As a bonus, a gif of rick floats around

Tech Used

HTML & CSS – layout and styling

JavaScript – redirect functionality

GIFs and images for added flair
